                       How to contribute to this website

   This website is maintained using
     * Markdown files for content
     * [10]Pelican (and theme notmyidea) to generate HTML
     * [11]pull requests against the [12]libexpat/libexpat.github.io
       repository on GitHub

   To render the website with your changes applied, running make should
   just work.

   If your contribution is about adding a new article, please:
    1. Add a new file content/pages/UPCOMING-TITLE.md
    2. Integrate links to that new aritcle at
    3. the news listing at content/pages/news.md
    4. the latest news section at the fron page content/pages/welcome.md

   We are looking forward to your [13]pull request!

   The Expat development team
